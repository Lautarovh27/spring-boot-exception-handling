# Spring Boot Error Handling

Proyecto desarrollado para practicar el manejo de errores y excepciones en Spring Boot.

## 🚀 Tecnologías utilizadas

- Java 21
- Spring Boot
- Maven
- Spring Web
- REST API

## 📚 Conceptos aplicados

- Manejo de excepciones personalizadas
- Uso de `@ExceptionHandler`
- Uso de `@ControllerAdvice`
- Respuestas HTTP personalizadas
- Manejo de errores globales
- Validación de datos
- Arquitectura en capas

## 📂 Estructura del proyecto

```text
src
├── controller
├── services
├── models
├── exceptions
└── SpringbootErrorApplication
```

## 🎯 Objetivo

Comprender cómo Spring Boot gestiona errores y excepciones dentro de una aplicación REST, implementando mecanismos para devolver respuestas claras y consistentes al cliente.


## 📖 Aprendizajes

Durante este proyecto se trabajó con:

- Creación de excepciones customizadas
- Centralización del manejo de errores
- Códigos de estado HTTP
- Buenas prácticas en APIs REST
- Inyección de dependencias
- Componentes y servicios de Spring
